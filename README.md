API на вход принимает парамтеры методом POST, обязательный параметр mode, в него могут быть переданы следующие значения:
register - для регистрации нового участника
list - просмотр ранее загруженных файлов
upload - загрузка нового файла

отвечам JSON формате, err - тег отвечающий за наличие какой либо ошибки, если он пустой или его нет, то успешна завершена работа API.

при mode=register в ответе получается только id который нужен для загрузки и просмотра ранее загруженных файлов

при mode=list обязательным становиться входной параметр id в который нужно передать ранее зарегистрированный id,
если не было загруженных изображений ранее то получите отшибку, если были то список с параметрами url, width, height

при mode=upload обязательным становиться входные параметры id, file_contents, width, height
id - ранее зарегистрированный id
file_contents - файл типа jpeg, gif, png
width - желаемая ширина
height - желаемая высота

в ответ при успехе список с параметрами url, width, height
